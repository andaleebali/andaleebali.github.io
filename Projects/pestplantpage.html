Pest Plant Dashboard

Role: GIS Analyst, Gisborne District Council

Project Summary

This interactive dashboard provides real-time visibility into pest plant surveillance and treatment efforts across the Gisborne region. It aggregates field-collected data into a clear and accessible map-based interface for council staff and stakeholders.

Key Contributions

  • Designed a user-friendly dashboard layout based on council needs
  • Built ArcGIS Dashboards linked to Field Maps inspections
  • Automated daily updates using ArcPy scripts
  • Enabled spatial filtering, treatment summaries, and mobile access

Tools & Technologies

  • ArcGIS Pro, ArcGIS Enterprise
  • Field Maps, Instant Apps, ArcGIS Dashboards
  • Python (ArcPy) for automation and data processing

Outcomes

The dashboard enhanced visibility over pest control efforts, allowing for faster response to problem areas and improved resource allocation. It has since been adopted across multiple departments for invasive species tracking.
